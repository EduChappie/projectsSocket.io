# Projetos com [Socket.io](https://socket.io) ⚡

## O que é Socket? 🤔
![](https://www.vectorlogo.zone/logos/socketio/socketio-ar21.png)
.
Para curiosos ou desconhecedores da tecnologia, [Socket.io](https://socket.io) é uma biblioteca de javascript foca na comunicação, com baixa latência ou baixo tempo de espera, bidirecional entre cliente e servidor.
Ótimo para pequenos trabalhos de troca de informações em *tempo real*

## Do que se trata o repostório? 📦
Como o nome sugere, aqui eu coloco projetos que eu utilizo essa tecnologia, mas não ficando limitado somente a essa biblioteca, aqui também posso usar outras bibliotecas ou ferramentas que o complementem ou que posso usar como estudo. Utilizando tecnologias como...
